# nasa-react-native

L’interface de cette application se compose :

[x] Un écran de lancement
[x] Une navigation avec une tabbar-La première tab s'affiche par défaut, elle contient la page d'accueil avec la photo du jour.
[x] La seconde tab la liste des photos du jour. 
[ ] Le tap sur une cellule affiche l’écran avec la photo. 
[ ] Un bouton dans la navbar permet de la partager.
[ ] la troisième tab est un formulaire de recherche de photos. 
[ ] Un bouton réinitialiser dans la navbar. 
[ ] En bas de page, le bouton rechercher permet d’afficher le résultat dans une nouvellepage.
